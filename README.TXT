--------------------------------------
Product names:

    Gramado Kernel - base/
    Gramado DE     - gramland/
    Gramado Games  - meta/

--------------------------------------
    base/     - (E) Boot loader, kernel and commands. 
    gramland/ - (A) Gramado Window System. (Window server and clients). (DE).
    meta      - (D) Games.
--------------------------------------
Build:
    Type 'make'
Clean up the mess:
    Go into the 'base/' directory and type 'make clean-all'
Running:
    Go into the base/ folder and type './run' or './runkvm'
--------------------------------------
Configuration file:
    new/include/config/superv/config.h
